// Scenario

NovaCorp's session token is a Base64-encoded serialized JSON object: {"username":"alice","role":"user","exp":...}. The server deserializes this token without signature verification. Decode the token, modify the role to admin, re-encode to Base64, and send it. The server will deserialize your crafted payload and grant admin access.
