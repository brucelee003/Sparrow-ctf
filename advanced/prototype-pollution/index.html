🔴 Advanced 300 pts Prototype Pollution

☣️ Prototype Pollution

A-04  ·  /advanced/prototype-pollution

// Scenario

NovaCorp's app uses a vulnerable deepMerge() function to process user-submitted JSON configuration. By injecting __proto__ or constructor.prototype keys into your JSON, you can pollute JavaScript's prototype chain — adding properties to every object in the runtime. The admin check later reads obj.isAdmin from an empty object. If the prototype is polluted, that check returns true.

⚙️ Config Merge Endpoint

Submit a JSON config object. The server merges it with the base config using deepMerge().

Admin Check — prototype chain inspection
Access control check (simplified):
const session = {}; // empty session object
if (session.isAdmin) { // reads from prototype if polluted!
  grantAdminAccess();
}
💡 Hint Try submitting JSON with a __proto__ key: {"__proto__": {"isAdmin": true}}. Then go to the "Verify Pollution" tab and run the admin check.
