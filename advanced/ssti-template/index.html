📌 SSTI Template

A-08  ·  /advanced/ssti-template

// Scenario

NovaCorp's email notification system generates greetings using a Jinja2-style template engine. User-supplied input is inserted directly into the template string before rendering: f"Hello, {name}!" → render_template("Hello, {{name}}!"). If the template engine interprets {{7*7}} as a mathematical expression rather than plain text, you've confirmed SSTI. From there, escalate to read server-side data.

📧 Notification Preview System
💡 Hint Start with {{7*7}} — if the output shows 49 instead of {{7*7}}, the template engine is evaluating your input. Then try {{config}} or {{flag}} to access server-side variables.
