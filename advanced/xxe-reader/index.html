📋 XXE Reader

A-03  ·  /advanced/xxe-reader

// Scenario

NovaCorp's XML import feature parses uploaded XML with an insecure parser that allows external entity processing. By defining an external entity that references a local file (file:///etc/passwd), you can make the XML parser read that file and include its contents in the parsed output. Craft an XXE payload and submit it.

XML Payload Editor
Template payloads:
HTTP Request Preview
POST /api/import-xml
POST /api/import-xml HTTP/1.1
Host: novacorp.com
Content-Type: application/xml
 
[XML body will appear here]

Send the XML first.

💡 Hint Define a DOCTYPE with an external entity: <!DOCTYPE foo [ <!ENTITY xxe SYSTEM "file:///etc/passwd" > ]> Then use &xxe; in your XML body. The parser will replace it with the file's contents.
