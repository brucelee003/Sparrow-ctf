🔍 Client-Side Check

B-08  ·  /beginner/client-side-check

// Scenario

NovaCorp's internal tools page is "protected" by a password prompt. The developer, in a hurry, implemented the check entirely in JavaScript — with the correct password hardcoded directly in the script file. The server never verifies anything. Open DevTools → Sources (or navigate directly to app.js) and find the password before entering it.
