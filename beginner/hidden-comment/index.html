💬 Hidden Comment

B-02  ·  /beginner/hidden-comment

// Scenario

During a late-night debugging session, one of NovaCorp's developers left a note to themselves inside the HTML source of this page. Something about credentials, or maybe a token. They pushed the commit and went to bed. Nobody reviewed it. The note is still there. You don't see it on the rendered page — but the source doesn't lie.

// Endpoint
Method URL Description
GET /beginner/hidden-comment/ This page — contains the hidden comment
// Steps
1
View the page source
Press Ctrl+U (or right-click → View Page Source)
2
Search for HTML comments
Use Ctrl+F and search for <!-- or FLAG
3
Enter the flag below
Copy the flag string and submit it to complete the challenge
💡 Hint HTML comments look like <!-- text here -->. They're invisible on the rendered page but fully visible in the raw source code.
