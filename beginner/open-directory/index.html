🟢 Beginner 50 pts Directory Listing

📂 Unprotected Directory

B-07  ·  /beginner/open-directory

// Scenario

NovaCorp's file upload service stores user files under /uploads/. The web server was never configured to disable directory listing for this folder. That means anyone who navigates to it sees a full list of every uploaded file — like browsing a shared hard drive. One of those files is particularly interesting.

// Endpoints
Method URL Description
GET /beginner/open-directory/uploads/ Open directory listing
GET /beginner/open-directory/uploads/secret_notes.txt Interesting file in the listing
// Steps
1
Navigate to the uploads directory
Browse: /beginner/open-directory/uploads/
2
Look at the file listing
Find the file that stands out from generic avatars and images
3
Open the interesting file
Click it to read its contents
💡 Hint Directory listing is enabled. Just navigate to the uploads/ folder and browse the available files.
