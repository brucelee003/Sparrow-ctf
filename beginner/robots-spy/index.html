🤖 Robots Spy

B-01  ·  /beginner/robots-spy

// Scenario

NovaCorp's sysadmin added a robots.txt file to stop search engines from crawling sensitive internal pages. Smart idea — except they listed every secret path they wanted hidden. Their attempts at security through obscurity just handed you a map. Find the disallowed path and visit it directly to retrieve what's there.

// Endpoints
Method URL Description
GET /robots.txt The robots exclusion file
GET /beginner/robots-spy/secret-admin-notes/ Disallowed path — contains the flag
// Steps
1
Open the robots.txt file
Navigate to: ../../robots.txt (root of this site)
2
Find the Disallow entry
Look for a path listed under Disallow: that looks sensitive
3
Visit that path directly
Navigate to the disallowed URL in your browser to retrieve the flag
💡 Hint robots.txt is always at the root of a web server. What paths are listed under Disallow:?
