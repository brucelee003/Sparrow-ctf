// Scenario

NovaCorp's product search takes your query and reflects it directly back into the page: "Results for: [your input]". No encoding, no sanitization. Whatever you type goes straight into the HTML. A classic reflected XSS scenario. Craft an input that causes the page to execute a script, and the system will reward you with a flag.
