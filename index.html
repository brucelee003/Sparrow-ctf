How It Works

🌐
Browser Only
All challenges are solvable using only your browser and DevTools. Burp Suite is optional for intermediate/advanced.
🚩
Flag Format
All flags follow the format FLAG{short_readable_text}. They appear when you successfully exploit the vulnerability.
📚
Educational Purpose
This platform is designed to teach real-world vulnerability classes in a safe, isolated environment. All scenarios are fictional.
⚡
Scoring
Beginner: 50 pts · Intermediate: 150 pts · Advanced: 300 pts. First solves earn bonus points. Max score: 4,000 pts.
🔒
Scope
Stay within this platform only. Do not attempt any techniques on real external systems. This is a learning-only environment.
💾
Progress Saved Locally
Your solved challenges are tracked in your browser's localStorage. Progress persists across sessions on the same device.
