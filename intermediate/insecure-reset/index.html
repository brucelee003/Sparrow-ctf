🔄 Insecure Reset Link

I-04  ·  /intermediate/insecure-reset

// Scenario

NovaCorp's "Forgot Password" feature sends a reset link with the target username in the URL: /reset?user=alice. The server never verifies that the person clicking the link is actually Alice — it just resets whatever username is in the parameter. You've received a reset link for alice. Change the parameter and reset the admin account instead.

// Endpoints
Method URL Description
GET ?user=alice Your legitimate reset link
GET ?user=admin Admin reset — returns recovery token (flag)
🔑 Password Reset Portal
Try different users:
?user=alice ?user=bob ?user=admin
💡 Hint Look at the URL bar. Change ?user=alice to ?user=admin and load the page.
