🔏 JWT Weak Secret

I-06  ·  /intermediate/jwt-weak

// Scenario

NovaCorp's API uses JWTs signed with HS256. The secret key is just secret. You've captured your own token as a regular user. Decode it, change "role":"user" to "role":"admin", re-sign it with the known weak secret, and present it to gain admin access.
